### Git vs ClawdHub 对比

| 维度 | Git 仓库 | ClawdHub |
|------|-----------|------------|
| **用户获取方式** | git clone | clawdhub install |
| **更新方式** | git pull（手动） | clawdhub update（半自动） |
| **依赖管理** | 需要手动 npm install | 自动安装依赖 |
| **技能发现** | 需要知道 Git URL | 可以搜索和发现 |
| **版本控制** | Git tags | ClawdHub 版本号 |
| **使用门槛** | 需要懂 Git | CLI 命令，简单 |
